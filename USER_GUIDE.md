# AI对话助手 - 用户使用指南

## 🎯 系统概述

AI对话助手是一个现代化的智能对话系统，提供以下核心功能：

- 🔐 **安全认证**: 用户注册/登录系统
- 💬 **智能对话**: 支持多种AI模型的实时对话
- 📚 **会话管理**: 自动保存和管理对话历史
- 🎨 **美观界面**: 响应式设计，支持移动端
- ⚡ **高性能**: 基于现代技术栈，响应迅速

## 🚪 开始使用

### 1. 访问系统
打开浏览器，访问：http://localhost:5173

### 2. 用户注册
首次使用需要创建账号：
- 点击"注册"标签
- 输入用户名（至少3个字符）
- 设置密码（至少6个字符）
- 确认密码
- 点击"注册"按钮

### 3. 用户登录
已有账号的用户：
- 点击"登录"标签
- 输入用户名和密码
- 点击"登录"按钮

## 💬 聊天功能

### 界面布局

```
┌─────────────────┬───────────────────────────────┐
│                 │  AI对话助手        用户名 [登出] │
│   历史对话      ├───────────────────────────────┤
│                 │                               │
│ • 新对话        │        对话消息区域            │
│ • 会话1         │                               │
│ • 会话2         │                               │
│                 │                               │
├─────────────────┼───────────────────────────────┤
│   [新对话]      │  [输入框] [模型选择] [发送]    │
└─────────────────┴───────────────────────────────┘
```

### 开始对话

1. **查看欢迎消息**
   - 登录后系统自动显示："Hi, 我能有什么能帮你的"

2. **输入问题**
   - 在底部输入框中输入您的问题
   - 支持多行输入（Shift+Enter换行）
   - 支持中英文混合输入

3. **选择AI模型**
   - DeepSeek Chat: 通用对话模型
   - DeepSeek Coder: 编程专用模型

4. **发送消息**
   - 点击"发送"按钮
   - 或按Enter键快速发送

5. **查看响应**
   - AI会自动分析并回复您的问题
   - 响应会实时显示在对话区域

### 对话示例

**用户**: 你好，请介绍一下自己
**AI**: 您好！我是基于DeepSeek的AI对话助手，请告诉我您需要什么帮助。

**用户**: 如何学习Python编程？
**AI**: 关于编程问题，我很乐意帮助您！您提到的'如何学习Python编程？'是一个很好的话题...

## 📚 会话管理

### 会话功能
- **自动创建**: 发送第一条消息时自动创建新会话
- **智能标题**: 根据首条消息自动生成会话标题
- **历史记录**: 左侧显示最近5个会话
- **快速切换**: 点击历史会话可快速切换

### 会话操作
- **新建会话**: 点击"新对话"按钮
- **选择会话**: 点击左侧会话列表中的任意会话
- **删除会话**: 点击会话右侧的"×"按钮
- **会话信息**: 显示创建时间和消息数量

## 🎨 界面特性

### 消息显示
- **用户消息**: 右侧显示，蓝色背景
- **AI消息**: 左侧显示，白色背景
- **头像标识**: 用户显示用户名首字母，AI显示"AI"标识
- **时间戳**: 每条消息显示发送时间

### 响应式设计
- **桌面端**: 完整的双栏布局
- **移动端**: 自适应单栏布局
- **触摸优化**: 支持触摸操作

### 状态指示
- **发送中**: 显示加载动画和"发送中"提示
- **思考中**: AI响应时显示"正在思考..."
- **错误提示**: 网络或服务错误时的友好提示

## ⚙️ 个人设置

### 用户信息
- **用户名显示**: 右上角显示当前登录用户
- **登出功能**: 点击"登出"按钮安全退出

### 模型选择
- **DeepSeek Chat**: 适合日常对话、问答、创作
- **DeepSeek Coder**: 适合编程问题、代码分析、技术讨论

## 🔧 高级功能

### 快捷键
- **Enter**: 发送消息
- **Shift+Enter**: 换行
- **Ctrl+/**: 聚焦输入框（计划中）

### 输入优化
- **自动调整**: 输入框高度自动调整
- **字符计数**: 显示输入字符数（计划中）
- **历史记录**: 输入历史记录（计划中）

## 🛡️ 安全与隐私

### 数据安全
- **加密存储**: 密码使用bcrypt加密
- **JWT认证**: 安全的身份验证机制
- **会话隔离**: 每个用户的数据完全隔离

### 隐私保护
- **本地存储**: 对话数据存储在本地服务器
- **访问控制**: 只有登录用户可以访问自己的数据
- **数据限制**: 最多保存5个会话，自动清理旧数据

## 📱 移动端使用

### 触屏操作
- **滑动**: 滚动查看历史消息
- **点击**: 选择会话和发送消息
- **长按**: 复制消息内容（计划中）

### 界面适配
- **自适应布局**: 根据屏幕尺寸自动调整
- **大按钮**: 适合触摸操作的按钮尺寸
- **简化导航**: 移动端优化的导航方式

## ❓ 常见问题

### Q: 忘记密码怎么办？
A: 目前需要重新注册账号，密码重置功能正在开发中。

### Q: 可以同时打开多个会话吗？
A: 当前版本一次只能查看一个会话，但可以快速切换。

### Q: AI响应速度慢怎么办？
A: 检查网络连接，或稍后重试。复杂问题可能需要更长处理时间。

### Q: 支持上传文件吗？
A: 当前版本只支持文本对话，文件上传功能正在开发中。

### Q: 如何删除账号？
A: 请联系管理员处理账号删除请求。
